工具

Web 字体生成器

  • Web Font Generator 根据 .ttf 或者 .otf 生成包含 .css, .eot, .woff, .svg 等文件的压缩包。
  • Create Your Own @font-face Kits 同上。

svg 转字体

  • IcoMoon App - Icon Font & SVG Generator 在线工具。可以上传 .svg 文件，生成包含 css、demo、web 字体的压缩包。
  • fontello/fontello 同上。
  • sapegin/grunt-webfont SVG to webfont converter for Grunt. Based on Font Custom.
  • iconfont CLI that converts Icon Font CSS StyleSheets into CommonJS modules exporting the unicodes by name.
  • Font Custom Generate custom icon webfonts from the comfort of the command line. 要求 ruby 环境。
