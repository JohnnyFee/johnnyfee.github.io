Setup

把 mongod 安装成服务时，用配置文件的方式貌似不行

mongod --config mongod.cfg --install

使用以下命令 OK

mongod --dbpath e:\mongo\data --logpath e:\mongo\log\mongo.log --install

之后就可以使用以下命令启动服务：

net start MongoDB

删除服务：

mongod --remove

停止服务：

net stop MongoDB

参考：Install MongoDB on Windows — MongoDB Manual 2.6.3
