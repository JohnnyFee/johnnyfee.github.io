CSS Layout

2014-07-11
  • CSS

常见的 Web 页面布局方式主要有：

  • 浮动布局（float）
  • 定位布局（position） 用于定位不用和其他元素发生交互的元素。
  • 行内块布局（inline-block）主要用于文字排版。
  • CSS3的多栏布局（Columns）
  • 伸缩布局（Flexbox）
  • 网格布局（Grids）只有 IE10+ 支持。
